# Predictive Urban Growth Model

## 📌 Overview
This project predicts high-growth real estate zones using machine learning and real-world housing data.

## ⚙️ Features
- Growth score prediction
- Feature engineering (price trends, rental yield, infrastructure proxy)
- Interactive Streamlit app
- Visualization of growth distribution

## 🧠 Model
- Random Forest Regressor

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
